bdScrollIntoView

  1. - directive in module scroll-into-view

If this directive's property evalutes to true, the element will be scrolled into view.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY
  bd-scroll-into-view="Boolean"
  bd-scroll-into-view-speed="Number">
...
</ANY>

Arguments

Param Type Details
bdScrollIntoView Boolean

Whether this directive should be active on the element
bdScrollIntoViewSpeed Number

How fast the content is scrolled

Examples

<input bd-scroll-into-view="true" />

<input bd-scroll-into-view="vm.someVariable" bd-scroll-into-view-speed="1500" />
