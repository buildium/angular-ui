buildium.angular-ui.compiledynamichtml.bdCompileDynamicHtml

  1. - directive in module

Replaces the contents of the element with the result of compiling the html string. Updates on changes to any contained scope variables

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY>
...
</ANY>

Example

<div bd-compile-dynamic-html="'<h1>{{title}}</h1>'"></div>
