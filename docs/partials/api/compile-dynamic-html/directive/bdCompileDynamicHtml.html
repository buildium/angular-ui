bdCompileDynamicHtml

  1. - directive in module compile-dynamic-html

Replaces the contents of the element with the result of compiling the html string. Updates on changes to any contained scope variables.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as attribute: 
    <ANY
  bd-compile-dynamic-html="string">
...
</ANY>

Arguments

Param Type Details
bdCompileDynamicHtml string

AngularJS expression evaluating to the html that should be rendered in this container.

Examples

<div ng-controller="ExampleController">
    <label>Title
        <input type="text" ng-model="title" />
    </label>
    <label>Dynamic Template
        <textarea ng-model="dynamicHtml" style="display: block"></textarea>
    </label>
    <h2>Result</h2>
    <div bd-compile-dynamic-html="dynamicHtml"></div>
</div>
angular.module('compileDynamicHtmlExample', [])
.controller('ExampleController', ['$scope', function($scope) {
    $scope.title = 'Hello world';
    $scope.dynamicHtml = '<b>{{title}}</b>'
}]);
