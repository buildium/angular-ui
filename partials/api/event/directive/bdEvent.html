bdEvent

  1. - directive in module event

Attach callbacks to arbitray DOM events.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-event
  bd-event="object">
...
</bd-event>
  • as attribute: 
    <ANY
  bd-event="object">
...
</ANY>

Arguments

Param Type Details
bdEvent object

A map of event names to functions on the scope to execute when that event is fired

Examples

<div bd-event="{ scroll: onScroll }"></div>
