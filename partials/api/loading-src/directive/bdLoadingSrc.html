bdLoadingSrc

  1. - directive in module loading-src

This directive will show a spinner and the message "Loading image" while an image is being downloaded. This is also useful when we are dynamically switching an images "src" attribute. If you use <img ng-src="{{some_src}}" />, there is odd behavior when you switch the image source. Specifically, the old image will display until the new one is loaded.

Directive Info

  • This directive executes at priority level 0.

Usage

  • as element: 
    <bd-loading-src
  bd-loading-src="string">
...
</bd-loading-src>
  • as attribute: 
    <ANY
  bd-loading-src="string">
...
</ANY>

Arguments

Param Type Details
bdLoadingSrc string

An angular expression that evaluates to the image url

Examples

<img title="Some Image" alt="Some Image" bd-loading-src="vm.image_src" />
